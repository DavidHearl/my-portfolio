Milestone Project 1

Website Link

This is the first project I had completed as a part of my code institute diploma. It was the final part of the HTML/CSS Essentials module

After some frustration using a friend's website (Link), I decided it would be a good idea to re-create it for my first project.

I tried to stick to the basic design and strucutre used in the original creation and improve on it by incorporating a responsive design. The previous design also included a form which needed to be downloaded and printed out, then filled in. This was a good opportunity to embed a form directly into the website.

Although this project was completed on 27/09/2021 it is my intension to revisit it. As I have progressed through the code institute course I have gained more knowledge and skills that can be used to improve certain aspects that I struggled with when I first started programming. My main focus will be in converting the existing layout into Bootstrap so It is easier to style and is more responsive.
